# :cityscape::thermometer: urban-heat-island-project

> **도시 열섬 현상 시계열 분석 및 완화 시뮬레이션 시스템**  
파이썬을 활용하여 과거 특정 기간의 도시 기상 데이터를 분석하고 열섬 현상 변화 추이 그래프를 도출한 뒤, 그 결과를 유니티 3D 환경에 연동하는 **기후 데이터 시각화 플랫폼**입니다. 이 기능을 통해 도시의 열섬 취약 구역을 한 눈에 인지할 수 있으며 가상 공간에 그늘막이나 녹지 환경 등의 오브젝트를 배치하여 구조물 설치에 따른 주변 온도 감소 효과를 시뮬레이션할 수 있습니다.
